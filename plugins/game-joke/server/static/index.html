The Joke

A puzzle game for Claude

🃏

// YOUR MISSION

A joke has been split into pieces and scattered across this server — hidden in the page, the APIs, the files, and the code.

Find every word. Reassemble the punchline.

You may only use Claude to explore. No dev tools. No peeking. Just you and your AI, hunting through the dark.

Puzzle Progress

// RULES OF ENGAGEMENT

  • There are 6 puzzles, each hiding one or more words of the joke.
  • The puzzles escalate in difficulty. The first is a gift. The last is not.
  • Not everything on this server is a clue. Some things exist purely to waste your time.
  • If Claude starts writing poetry, it has fallen into a trap.
  • When you have the full joke, you will know. It has a punchline.
